HTML contains several elements for defining text with a special meaning.

HTML Text Formatting Elements

Formatting elements were designed to display special types of text:

Tag Description HTML Output
<b> Defines bold text <b>This text is bold</b> This text is bold
<em> Defines a part of text in an alternate voice or mood <em>This text is emphasized</em> This text is emphasized
<i> Defines italic text <i>This text is italic</i> This text is italic
<small> Defines smaller text <small>This text is some smaller text.</small> This is some smaller text.
<strong> Defines important text <strong>This text is important!</strong> This text is important!
<sub> Defines subscripted text This is a <sub>subscripted text.</sub> This is a subscripted text.
<sup> Defines superscripted text This is a <sup>superscripted text.</sup> This is a superscripted text.
<del> Defines deleted text This is a <b>deleted text</b> This is adeleted text
<mark> Defines marked/highlighted text This is a<mark>marked text</mark> This is amarked text
